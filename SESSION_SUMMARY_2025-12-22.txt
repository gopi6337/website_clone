HOSTING & DEPLOYMENT INFORMATION
============================================================

Ready for Static Hosting:

Recommended Platforms:
1. Vercel (Recommended) - FREE
   - Automatic builds from Git
   - Global CDN
   - Custom domain support
   - SSL certificate included

2. Netlify - FREE
   - Similar features to Vercel
   - Easy GitHub integration

3. Cloudflare Pages - FREE
   - Unlimited bandwidth
   - Fastest CDN

Deployment Requirements:
- Build command: pnpm build
- Output directory: dist/public
- Node.js: 18+ or 20+
- No database required
- No backend server needed (static hosting)

Email Integration:
- Service: Formspree (FREE tier: 50 submissions/month)
- Setup required: Create account, get form ID
- Configuration: Update formspreeEndpoint in BookingForm.tsx
- Documentation: FORMSPREE_SETUP_GUIDE.md
